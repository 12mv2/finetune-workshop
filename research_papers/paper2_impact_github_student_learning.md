# The Impact of GitHub on Students' Learning and Engagement in a Software Engineering Course

## Authors
Prutha Patani, Saurabh Tiwari, Santosh Singh Rathore

## Publication Details
- **Published:** 2024
- **Journal:** Computer Applications in Engineering Education (Wiley)
- **DOI:** https://doi.org/10.1002/cae.22775
- **SSRN:** https://papers.ssrn.com/sol3/papers.cfm?abstract_id=4264284

## Overview
This study examines the effectiveness of GitHub in improving student learning outcomes, project development skills, and competence in version control systems within a software engineering course context.

## Key Findings

### Student Competence Improvement
The research demonstrates that students reported significant improvements in GitHub competence after using the platform for coursework. This finding aligns with previous studies by:
- Tushev et al. (2020)
- Bennedsen et al. (2022)
- Patani et al. (2024)

### Industry-Relevant Skills
GitHub provides students with:
- **Industry Tool Proficiency** - Experience with tools commonly used in professional software development
- **Collaboration Opportunities** - Enhanced teamwork and distributed collaboration skills
- **Self-Directed Learning** - Increased autonomy in managing their own learning process

### Engagement Patterns
Students engage with GitHub repositories through multiple mechanisms:
- Reviewing change logs
- Commenting on code
- Collaborating in forums and discussion threads
- Asking instructor questions through issue tracking
- Using and building upon each other's code
- Reporting error messages and bugs

## Educational Benefits

### 1. Project Development Skills
Students develop practical skills in:
- Version control workflows
- Code review processes
- Project documentation
- Issue tracking and management

### 2. Collaborative Learning
GitHub facilitates:
- Peer-to-peer learning
- Code sharing and reuse
- Constructive feedback through pull requests
- Team-based software development

### 3. Professional Preparation
The platform bridges academic and professional environments by:
- Simulating real-world development workflows
- Building portfolios of work
- Demonstrating competence to potential employers

## Implications for Educators

This research suggests that integrating GitHub into software engineering courses:
- Enhances student engagement with course material
- Develops industry-relevant technical skills
- Promotes collaborative learning practices
- Prepares students for professional software development careers

## Conclusion

The study provides evidence that GitHub serves as an effective educational platform for software engineering education, improving both technical competence and student engagement when properly integrated into coursework.
