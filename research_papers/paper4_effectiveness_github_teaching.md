# An Investigation Into the Perceived Effectiveness of GitHub Repositories to Teach Programming

## Author
**Ebrahim Adam** - Head of Computer Science at The Independent Institute of Education, Varsity College, Durban, South Africa

## Publication Details
- **Published:** November 21, 2024
- **Conference:** 1st International Conference on Education Research (Vol. 1 No. 1)
- **DOI:** https://doi.org/10.34190/icer.1.1.2774
- **Full Paper URL:** https://papers.academic-conferences.org/index.php/icer/article/view/2774

## Overview
This paper examines how lecturer-created GitHub repositories support programming education using constructivist principles, where students learn concepts through structured examples before tackling complex projects.

## Research Focus
The study investigates the perceived effectiveness of GitHub repositories as teaching tools in programming education, analyzing how structured repository examples help students understand concepts progressively before engaging with complex programming challenges.

## Main Findings

### Advantages Identified:
1. **Enhanced Collaboration** - GitHub repositories facilitate team-based learning and peer collaboration
2. **Self-Directed Learning** - Students can explore code examples at their own pace
3. **High Effectiveness Rating** - Students rated lecturer-provided repositories with strongly positive feedback (μ: 4.38, M: 5.0, s: 0.78)

### Challenges Identified:
1. **Team Coordination Difficulties** - Managing collaborative work in distributed teams
2. **Technical Obstacles** - Connectivity issues and access problems
3. **Learning Curve** - Initial difficulty understanding Git/GitHub workflows

## Recommendations for Improvement

The research provides several key recommendations for enhancing GitHub repository effectiveness in education:

1. **Better Code Documentation**
   - Include more comprehensive code comments
   - Explain complex logic and design decisions

2. **Visual Aids**
   - Incorporate images and diagrams
   - Add video tutorials and walkthroughs

3. **Structured Repository Organization**
   - Clear folder hierarchy
   - Progressive difficulty levels
   - Well-organized README files

4. **Academic Adoption Strategies**
   - Standardized repository templates for courses
   - Integration with learning management systems
   - Clear guidelines for student contributions

## Implications for Educators

This study demonstrates that well-structured GitHub repositories significantly enhance programming education when designed with clear structure, comprehensive documentation, and multimedia resources. Educators should focus on repository organization and documentation quality to maximize student learning outcomes.
